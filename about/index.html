Info

Diese Seite dient zum auflisten aller Tools die ich benutzt habe um diese Website zu erstellen.

Haikei

Haikei ist eine einfach bedienbare Website mit der man komplexe SVG-Grafiken schnell und einfach erstellen kann.

Zum Interagieren einfach mit der Maus drüber hovern.

Shape Divider App

Genau wie Haikei dient diese Website zum Erstellen von SVG-Grafiken, hier aber insbesondere mit dem Fokus auf die Gestaltung von Dividern.

Zum Interagieren einfach mit der Maus drüber hovern.

Bootstrap Icons

Auf Bootstrap Icons findet man professionelle, leicht zu benutzende Icons, die dank SVG-Grafik ohne Qualitätsverlust ganz groß skaliert werden können.

Zum Interagieren einfach mit der Maus drüber hovern.

Recherche

Desweiteren waren Fireship und die MDN Web Docs eine große Hilfe zum nachlesen und um sich Dinge erklären zu lassen.
